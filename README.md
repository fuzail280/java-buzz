# Java Buzz Coffee Shop Website

This is a coffee shop website project called **Java Buzz**.  
The website shows a coffee menu, blog section, location map, and an order form.

## Features
- Home section with coffee images
- About section
- Coffee menu
- Blog section (JSON data)
- Order form
- Location map
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- Swiper.js
- Leaflet Map

## SEO
- Meta title, description, and keywords added
- sitemap.xml created
- robots.txt created
